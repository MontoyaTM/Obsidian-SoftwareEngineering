# Q.2: What is Inheritance? Why is it important?

Inheritance is a fundamental concept of OOP that allows for the creation of a new class based on an existing one while inheriting the members and behavior of the parent class.

It allows for the reusability and abstraction of code.  

# Q.3: What are the different types of Inheritance? 

- Single Inheritance
- Multi Inheritance (Interfaces)
- Hierarchical Inheritance
- Multilevel Inheritance
