> # Core Concept 

This note demonstrates how to pass data between Blazor WebAssembly pages using the browser's **Session Storage** via JavaScript interop (`IJSRuntime`).

- Rather than using a shared service or state container, data is persisted temporarily in the **browser's native `sessionStorage`** — then retrieved on the next page. This is done by calling JavaScript APIs directly from C# using `IJSRuntime`.

### How It Works (Two-Page Flow)

**Step 1** captures user input and stores it before navigating away:

``` run-csharp
JSRuntime.InvokeVoidAsync("sessionStorage.setItem", "demo-data", data);
NavigationManager.NavigateTo("/browser_wasm_2");
```


**Step 2** retrieves it after the component has fully rendered:

``` run-csharp
data = await JSRuntime.InvokeAsync<string>("sessionStorage.getItem", "demo-data");
StateHasChanged(); // Triggers re-render to display the retrieved value
```

### Why `OnInitializedAsync()` Cannot Call JSRuntime

`OnInitializedAsync()` fires **before the component is rendered to the DOM**. At this point in WASM:

- The browser has not yet painted any elements
- JavaScript cannot interact with DOM nodes that don't exist yet
- `sessionStorage.getItem` via `IJSRuntime` will **silently fail or throw** because the JS interop bridge isn't fully ready

This is why `OnInitializedAsync()` is left as a base call only in the example — it's intentionally empty for JS interop purposes.

# Client: BrowserStorageStep1.razor


``` run-csharp
@page "/browser_wasm_1"

@rendermode InteractiveWebAssembly

@inject IJSRuntime JSRuntime
@inject NavigationManager NavigationManager

<h3>BrowserStorageStep1</h3>
<br/>

<input type="text" class="form-control" @bind-value="data"/>
<br />

<button type="button" class="btn btn-primary" @onclick="GoNext">Next</button>

@code {
    private string data = string.Empty;

    private void GoNext()
    {
        JSRuntime.InvokeVoidAsync("sessionStorage.setItem", "demo-data", data);
        NavigationManager.NavigateTo("/browser_wasm_2");
    }
}
```


# Client: BrowserStorageStep2.razor

``` run-csharp
@page "/browser_wasm_2"

@rendermode InteractiveWebAssembly

@inject IJSRuntime JSRuntime

<h3>BrowserStorageStep2</h3>
<br />

<b>Demo Data: </b> @data

@code {
    private string data = string.Empty;

    protected override async Task OnInitializedAsync()
    {
        await base.OnInitializedAsync();
    }

    protected override async Task OnAfterRenderAsync(bool firstRender)
    {
        if(firstRender)
        {
            data = await JSRuntime.InvokeAsync<string>("sessionStorage.getItem", "demo-data");
            StateHasChanged();
        }
    }
}
```
