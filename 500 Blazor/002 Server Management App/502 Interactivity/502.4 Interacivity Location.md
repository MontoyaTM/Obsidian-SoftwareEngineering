# Page/Component

## Server.razor

##### Enable InteractiveServer on the Component Initialization (Recommended)

``` run-csharp
<ServerComponent @rendermode="InteractiveServer"></ServerComponent>
```

##### Enable InteractiveServer on the Parent Component

``` run-csharp
@page "/servers"

@rendermode InteractiveServer
@inject NavigationManager NavigationManager
```

## ServerComponent.razor

##### Enable InteractiveServer on the Reusable Component

``` run-csharp
@rendermode InteractiveServer
```

# Global

## App.razor

``` run-csharp
<Routes @rendermode="InteractiveServer"/>
```
