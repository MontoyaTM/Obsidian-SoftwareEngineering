# SearchBarComponent.razor

``` run-csharp
<div class="input-group mb-3">
    <input type="text" class="form-control" placeholder="Search Servers"
           @bind-value="serverFilter"
           @bind-value:event="oninput" />
    <button class="btn btn-outline-secondary" type="button" id="button-search" @onclick="HandleSearch">Search</button>
</div>

@code {
    [Parameter]
    public EventCallback<string> SearchServerCallback { get; set; }

    private string serverFilter = string.Empty;

    private void HandleSearch()
    {
        SearchServerCallback.InvokeAsync(serverFilter);
    }
}
```
