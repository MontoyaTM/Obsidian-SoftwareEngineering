# Routable Component

Routable components are components that can be navigated to via a URL. They are defined using the <span style="color:rgb(181, 98, 249)">@page</span> directive

### Key traits

- Have their own **route (URL)**
- Are discovered by the **Router (`Routes.razor`)**
- Typically represent **full pages**

``` run-csharp
@page "/servers"
```


# Non-routable Component

Routable components are components Components that **cannot be accessed directly via a URL**.

### Key traits

- Reusable building blocks
- Used for UI composition
- Not visible in routing
