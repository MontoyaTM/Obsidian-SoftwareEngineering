>  # Two Sum

Given an array of numbers <span style="color:#81fd83">sorted</span> in ascending order and a target sum, find aÂ **pair in the array whose sum is equal to the given target**.

Write a function to return the indices of the two numbers (i.e. the pair) such that they add up to the given target. If no such pair exists return (-1, -1).

``` run-csharp
public int[] TwoSum(int[] nums, int target)
{ 
	int i = 0, j = nums.Length - 1;
	
	while(i < j)
	{
		int sum = nums[i] + nums[j];
		
		if(sum == target)
		{
			return new int[] {i, j};
		}
		
		if(sum > target)
		{
			j--;
		} else
		{
			i+;
		}
	}
	
	return new int[] {-1, -1};
}
```
