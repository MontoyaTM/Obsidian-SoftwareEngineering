>  # Hash Map

![[Pasted image 20250318152138.png]]

### Hashing: 
   
   It's a technique that takes an input (or 'message') and returns a fixed-size string, which looks random. The output, known as the hash value, is unique (mostly) to the given input.

### Hash Function: 

   This is our "magical spell" that converts the input data (like our book title) into a fixed-length value.

However, no magic is perfect. If two different inputs give the same hash, that's called a **collision**. It's like our spell accidentally pointing to the same spot for two different books.
