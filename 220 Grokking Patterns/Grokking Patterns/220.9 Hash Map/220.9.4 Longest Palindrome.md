> [!note] 
>  # Longest Palindrome

Given a string, determine the length of the longest palindrome that can be constructed using the characters from the string. You don't need to return the palindrome itself, just its maximum possible length.

![[Pasted image 20250324103859.png]]

> [!success] 
> ## Solution 

To solve this problem, we can use a hashmap to keep track of the frequency of each character in the string. The idea is to use pairs of characters to form the palindrome.

   - For example, if a character appears an even number of times, we can use all of them in the palindrome.

   - If a character appears an odd number of times, we can use all except one of them in the palindrome.

   - Additionally, if there's any character that appears an odd number of times, we can use one of them as the center of the palindrome.

![[Pasted image 20250324113408.png ]]


> [!example] 
>  ## Code Snippet


``` run-csharp
public int longestPalindrone(string str)
{
	Dictionary<char, int> freq = new Dictionary<char, int>();
	int length = 0;
	bool oddFound = false;
	
	foreach(char c in str)
	{
		freq[c] = freq.GetValueOrDefault(c, 0) + 1;
	}
	
	foreach(int freq in freq.Values)
	{
		if(freq % 2 == 0)
		{
			length += freq;
		}
		else
		{
			length += freq - 1;
			oddFound = true;
		}
	}
	
	if(oddFound)
		length++;
	
	return length;
}
```
