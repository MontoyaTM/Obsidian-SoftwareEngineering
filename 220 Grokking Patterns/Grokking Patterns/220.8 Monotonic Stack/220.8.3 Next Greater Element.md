> [!note] 
>  # Next Greater Element

Given two integer arrays `nums1` and `nums2`, return an array `answer` such that `answer[i]` is the next greater number for every `nums1[i]` in `nums2`.

![[Pasted image 20250313213602.png]]

> [!success] 
> ## Solution 

This problem can be solved using a Monotonic Stack and Dictionary. The Monotonic Stack helps to find the next greater element for each element in `nums2`. The Dictionary then maps each element to its next greater element.

![[Pasted image 20250313213831.png]]

> [!example] 
>  ## Code Snippet

``` run-csharp
public int[] nextGreaterElement(int[] nums1, int[] nums2)
{
	Stack<int> stack = new Stack<int>();
	Dictionary<int, int> dict = new Dictionary<int, int>();
	
	foreach(int num in nums2)
	{
		while(stack.Count > 0 && stack.Peek() < num)
			dict[stack.Pop()] = num;
		
		stack.Push(num);
	}
	
	for(int i = 0; i < nums1.Length; i++)
	{
		nums1[i] = dict.GetValueOrDefault(nums1[i], - 1);
	}
}
```
