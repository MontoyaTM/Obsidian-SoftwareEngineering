# 100 CSharp Fundamentals

```base
properties:
  file.name:
    displayName: Tips
  note.tags:
    displayName: Tags
views:
  - type: table
    name: CSharp Fundamentals
    filters:
      and:
        - file.inFolder("100 CSharp Fundamentals")
    order:
      - file.name
      - tags
    sort:
      - property: file.name
        direction: ASC
      - property: tags
        direction: ASC
    columnSize:
      file.name: 955

```
