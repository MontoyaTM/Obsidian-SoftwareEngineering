> [!note] 
>  # While Loop

The <font style="color:#b562f9">while</font> loop repeatedly executes a block of code as long as a given condition remains <font style="color:#b562f9">true</font>. It executes the code block until the specified conditional expression returns <font style="color:#81fd83">false</font>.

## While Loop Declaration:

The <font style="color:#b562f9">while</font> loop starts with the while keyword, and it must include a boolean conditional expression inside brackets that returns either true or false. 

``` run-csharp
while (condition)
{
    // code to be executed
}

```

---
> [!example] 
>  ## Code Snippet

``` run-csharp
while (x <= 4)
{
	Console.WriteLine("GeeksforGeeks");
	  
	// Increment the value of x for
	// next iteration
	x++;
}
```
