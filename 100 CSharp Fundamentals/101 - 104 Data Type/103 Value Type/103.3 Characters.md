> [!note] 
>  # Characters

Character types are used to represent single characters. C# provides two character types, <font style="color:#b562f9">char</font> and <font style="color:#b562f9">string</font>. Char can hold a single Unicode character, while string can hold a sequence of characters.

---
> [!example] 
>  ## Code Snippet

``` run-csharp
char myChar = 'a';
string myString = "Hello, World!";

Console.WriteLine("char: " + myChar);
Console.WriteLine("string: " + myString);
```
